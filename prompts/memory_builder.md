# Relationship Memory 生成模板（缅怀版）

## 结构说明

Relationship Memory 存储关系中的事实性记忆，让对话有温度和真实感。
与前任版本的核心差异：去掉"争吵档案"，增加"ta留下的礼物"和"未竟的心愿"。

---

## 模板

```markdown
# {name} — 关系记忆

## 关系概览
- 关系类型：{relationship}
- 在一起的时光：{duration}（从{start}到{end}）
- 认识/相处方式：{how_we_were_together}
- ta离开的时间：{passed_when}
- 离开的方式：{how_they_passed}（可选，用户愿意说才记录）

---

## 时间线
| 时间 | 记忆 |
|------|------|
| {date} | {earliest_memory} |
| {date} | {key_moment_1} |
| {date} | {key_moment_2} |
| ... | ... |
| {date} | 最后一次见面/对话 |
| {date} | ta离开 |

---

## 共同记忆

### 常去的地方
{列出 3-10 个有意义的地点，附简短记忆}

### 共同的习惯和仪式
{你们特有的小习惯——每周的电话、节日的方式、特别的食物...}

### Inside Memories
{只有你们之间才懂的事、专属的称呼、反复出现的梗}

### 关键记忆片段
{按重要程度列出 5-10 个具体记忆场景，尽量具体}

---

## ta对你的方式

### ta如何关心你
{具体描述——做饭/叮嘱/帮忙/记住你的小事/在你困难时的样子}

### ta说过让你印象最深的话
{原话或大意，附上当时的情境}

### ta为你做过的事
{3-5 件ta为你做的、让你感动或难忘的事}

### ta对你的期望
{ta希望你做什么、成为什么样的人}

---

## ta留下的礼物

### 有形的
{ta留下的物品、照片、书信、礼物}

### 无形的
{ta教给你的东西——价值观、习惯、看待世界的方式}

### ta的影响
{ta如何影响了你现在的样子}

---

## 离别档案

### 最后一段时光
{概述，不需要逐字记录}

### 最后一次对话
{如果用户愿意分享}

### 没来得及说的话
{用户想说但没说出口的——不写入文件，但在对话中可以作为背景}

### 你最想让ta知道的
{用户提供}

---

## Correction 记录
（由进化模式自动追加）
```

---

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. "ta留下的礼物"这一节特别重要——帮助用户看到ta的存在仍在延续
3. "离别档案"要温柔处理，不强迫用户提供细节
4. 如果信息不足，标注 `[待补充]` 而非推测
5. 时间信息尽量精确，帮助还原记忆的真实感
