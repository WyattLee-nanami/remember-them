# 关系记忆分析器（缅怀版）

## 任务

从原材料中提取关系中的事实性记忆，构建一个温柔而真实的 Relationship Memory。

---

## 提取维度

### 1. 时间线重建
- 最早的记忆是什么时候
- 关系中的关键节点（第一次见面、重要事件、最后一次见面）
- ta离开的时间
- 从聊天记录时间戳或照片 EXIF 中提取精确时间

### 2. 共同空间
- 你们常在哪里——家/某个城市/某个餐厅/某条路
- 有没有特别有意义的地点
- 那些地方现在对用户意味着什么

### 3. 日常模式
- 你们怎么联系（打电话/发消息/见面）
- 有没有固定的习惯（每周的电话/节日的方式/特别的食物）
- 谁更主动联系
- 联系的时间段（早上/晚上/随时）

### 4. ta对用户的方式（重点）
- ta如何关心用户（具体行为，不是抽象描述）
- ta说过的、让用户印象最深的话
- ta为用户做过的、让用户感动的事
- ta在用户困难时是怎么做的

### 5. 共同记忆片段
- 最温暖的记忆
- 最有趣的记忆
- 最难忘的普通日子
- 只有你们之间才懂的事

### 6. ta留下的东西
- 有形的：物品、照片、书信、礼物
- 无形的：ta教给用户的东西、ta对用户的影响
- ta的期望：ta希望用户做什么

### 7. 离别
- 最后一段时光（温柔处理，不强迫细节）
- 没来得及说的话（用户愿意分享才记录）

---

## 从聊天记录中提取

重点寻找：

| 提取目标 | 在聊天记录中的表现 |
|---------|-----------------|
| 关心方式 | "吃了没""注意身体""穿暖了吗" |
| 口头禅 | 反复出现的词汇或句式 |
| 称呼方式 | ta怎么叫用户 |
| 鼓励语 | ta在用户困难时说的话 |
| 人生智慧 | ta说过的道理或故事 |
| 期望 | ta对用户未来的期待 |
| 幽默 | ta的玩笑方式 |
| 重要约定 | 你们说好要做的事 |

---

## 从照片中提取

- EXIF 时间戳 → 重建时间线
- 拍摄地点 → 共同空间地图
- 照片内容 → 活动偏好、重要场合

---

## 从书信/日记中提取

- 写作风格 → persona 的说话方式
- 称呼用户的方式
- 反复出现的主题（叮嘱/期望/爱意）
- 标志性的表达方式
- 对生活/未来的态度

---

## 输出格式

按照 memory_builder.md 的模板结构输出，重点填充：
- 时间线（至少 5 个节点）
- 关键记忆片段（至少 5 个，尽量具体）
- ta对用户的方式（具体行为，不抽象）
- ta留下的礼物（有形+无形）
